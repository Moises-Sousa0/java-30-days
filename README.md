# Java 30 Days

Projeto pessoal para retomar os estudos em Java e construir consistência.
A ideia é desenvolver um sistema simples no terminal (CLI), evoluindo aos poucos.

## Objetivo
- Voltar a estudar programação de forma prática
- Construir algo real, mesmo simples
- Manter commits frequentes no GitHub

## Regras
- Trabalhar no projeto quase todos os dias
- Focar em Java como linguagem principal
